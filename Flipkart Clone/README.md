# Flipkart Clone — E-Commerce Platform

A full-stack e-commerce web application replicating Flipkart's design and user experience.

## Tech Stack
- **Frontend:** React.js (Create React App)
- **Backend:** Node.js + Express.js
- **Database:** PostgreSQL

## Prerequisites
- Node.js >= 18
- PostgreSQL >= 14
- npm or yarn
